# en_vocabulary_learning
英単語学習を効率化できるアプリです。

## URL
### フロントエンドURL
https://en-vocabulary-learning.netlify.app

### バックエンドURL
https://en-vocabulary-learning-api.herokuapp.com

## 機能一覧
### 認証関連
- サインイン
- ログイン
- ログアウト

### メイン機能
- 単語の登録・更新
  - 英英辞書の外部API( https://www.wordsapi.com )と連携し、単語それぞれの定義と用法を複数取得し、登録します。
- 削除・一括削除
- 発音機能
- 単語の理解度のみの更新
